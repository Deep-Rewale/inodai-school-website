# Inodai Waldorf School Website

## Project Description
The **Inodai Waldorf School Website** is a modern and responsive educational website developed to present information about the school, its programs, philosophy, events, and admissions.

This project focuses on providing a clean, professional, and user-friendly interface for parents and visitors while maintaining a premium educational design.

## Tech Stack
- React (Vite)
- Tailwind CSS
- React Router
- Framer Motion
- JavaScript (ES6)
- HTML5
- CSS3

---

## Features
- Fully responsive design
- Modern UI for educational website
- Multi-page navigation using React Router
- Animated UI components
- Programs section (Playgroup, Kindergarten, Daycare)
- Waldorf Education information pages
- Events section
- Board of Directors page
- Contact page
- Admissions login page
- Dropdown navigation menu

## Installation & Setup

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/Deep-Rewale/inodai-school-website.git
```

### 2. Navigate to Project Folder

```bash
cd inodai-school-website
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run the Development Server

```bash
npm run dev
```

### 5. Open in Browser

Open the following URL in your browser:

```
http://localhost:5173
```
